Aunties Recipes is a free search engine for all the BBC Food recipes. It's a recipe archive built by @user24 as a fun side project when he's not programming garden planning software at work.

FAQ: What if the BBC moves or removes the recipes?
A: I have archived all the data so I could update links or rehost them if necessary :)
